# LJD Training - ระบบจองคลาสเรียน (PT & Group Class Booking Platform)

แพลตฟอร์มจำลองระบบจองคลาสเรียนส่วนตัว (1-on-1 PT) และคลาสกลุ่ม (Group Class) สำหรับผู้เรียน เทรนเนอร์ และผู้ดูแลระบบ พัฒนาด้วยเทคโนโลยีเว็บสมัยใหม่ที่สวยงามและใช้งานง่าย (Dark Glassmorphism)

---

## 🚀 วิธีการเปิดใช้งานระบบบนเครื่องของคุณ (How to Run)

คุณสามารถเปิดใช้งานระบบบนเครื่อง Mac ของคุณได้ง่ายๆ ผ่าน 2 วิธีดังนี้:

### วิธีที่ 1: ดับเบิ้ลคลิกเพื่อเริ่มต้น (ง่ายที่สุด)
1. เปิดโฟลเดอร์โครงการ `AI Agent` ใน Finder
2. ดับเบิ้ลคลิกที่ไฟล์ **`start.sh`**
3. ระบบจะเปิดเว็บเบราว์เซอร์พาคุณไปยังหน้าแรก (`http://localhost:8000`) และเริ่มต้นเซิร์ฟเวอร์จำลองให้คุณทันที!

### วิธีที่ 2: รันผ่าน Terminal
1. เปิด Terminal และนำทางไปยังโฟลเดอร์ของโครงการ
2. รันคำสั่งต่อไปนี้:
   ```bash
   ./start.sh
   ```

## 💡 ฟีเจอร์หลักของระบบ (Key Features)

ระบบจำลองบทบาทผู้ใช้งานไว้ทั้งหมด 3 มุมมอง สามารถสลับเล่นได้ทันทีที่มุมขวาบนของหน้าจอ:

### 1. มุมมองผู้เรียน (Student View)
- **Remaining Credits**: แสดงจำนวนเครดิตหรือครั้งคงเหลือแยกตามแพ็กเกจของตนเอง
- **Your Next Session**: แสดงรายละเอียดคลาสเรียนถัดไปที่กำลังจะมาถึง พร้อมปุ่มยกเลิกการจองเพื่อคืนสิทธิ์ (Refund)
- **Book a Training Session**: จองคลาสเรียนแบบ Private (1 on 1) หรือคลาสกลุ่ม (Group Class) แบบ Real-time ตัวกรองประเภทคลาส และหลอดแสดงสถานะความจุคลาส
- **My Bookings**: ตารางสรุปรายการคลาสทั้งหมดที่ผู้เรียนคนนี้จองไว้

### 2. มุมมองเทรนเนอร์ (Trainer View)
- **Trainer Dashboard**: แสดงสถิติจำนวนชั่วโมงสอนทั้งหมด แยกตามประเภทคลาสส่วนตัวและคลาสกลุ่ม
- **Create Session**: เพิ่มตารางสอนใหม่ ระบุเทรนเนอร์ ประเภทคลาส จำนวนคนสูงสุด วัน และเวลาสอน
- **View Participants**: คลิกเพื่อดูรายชื่อนักเรียนที่จองคลาสเรียนนั้นๆ รวมถึงช่องทางติดต่อ (เบอร์โทรและอีเมล)

### 3. มุมมองผู้ดูแลระบบ (Admin View)
- **Students & Packages**: แสดงตารางรายชื่อผู้เรียนทั้งหมดในระบบ พร้อมสิทธิ์การปรับแต่ง/เพิ่มเครดิต (Adjust Credits) ได้ตามต้องการ
- **All Sessions**: สรุปตารางคลาสเรียนทั้งหมด และดูรายชื่อนักเรียนที่จองในคลาสนั้นๆ ได้ทันที
- **Data Export**: ปุ่มส่งออกข้อมูลแต่ละตาราง (Users, Sessions, Bookings, Packages) ออกมาเป็นไฟล์ **CSV (UTF-8 BOM)** เพื่อนำไปเปิดใน Microsoft Excel หรือนำไปวางลงใน Google Sheets
- **Database Utilities**: ปุ่มรีเซ็ตข้อมูลทั้งหมดกลับเป็นข้อมูลตั้งต้นเพื่อใช้ทดสอบใหม่

## 🌐 การนำขึ้นระบบออนไลน์ / โฮสติ้ง (Deployment Guide)

เนื่องจากระบบนี้เป็นเว็บไซต์ประเภท **Static Web Application** (ทำงานบน Browser และบันทึกข้อมูลจำลองไว้ที่ LocalStorage) คุณจึงสามารถนำไปอัปโหลดขึ้นโฮสติ้งฟรีเพื่อใช้งานจริงหรือแชร์ให้ผู้อื่นทดลองใช้งานได้อย่างง่ายดาย:

### 1. โฮสต์ด้วย Netlify (แนะนำ - ง่ายที่สุด)
1. สมัครใช้งานเว็บ [Netlify](https://www.netlify.com/)
2. ไปที่หน้าหลักของ Dashboard เลื่อนลงมายังส่วน **Drag and drop your site folder here**
3. ลากโฟลเดอร์โครงการ `AI Agent` (ที่มีไฟล์ `index.html`, `app.js`, `styles.css`) ไปวาง
4. เสร็จสิ้น! คุณจะได้รับลิงก์ URL (เช่น `https://random-name.netlify.app`) สำหรับส่งให้ผู้ใช้จองคลาสเรียนได้ทันที

### 2. โฮสต์ด้วย GitHub Pages
1. สร้าง Repository ใหม่บน GitHub และอัปโหลดไฟล์โครงการขึ้นไป
2. ไปที่เมนู **Settings** > **Pages** ของ Repository นั้นๆ
3. ในส่วน **Build and deployment** > ให้เลือก Source เป็น **Deploy from a branch** และเลือกสาขาหลัก (เช่น `main`)
4. กดบันทึก ระบบจะสร้าง URL ฟรีให้คุณภายใน 1-2 นาที
