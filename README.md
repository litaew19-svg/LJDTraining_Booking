# 🏋️‍♂️ LJD Training - PT & Group Class Booking Platform

ระบบบริหารจัดการตารางจองคลาสเรียนส่วนตัว (1 on 1 Personal Training) และคลาสกลุ่ม (Group Class) สำหรับ **LJD Training** รูปแบบเว็บแอปพลิเคชันพรีเมียม (Premium Dark Glassmorphism) พร้อมจำลองการจัดเก็บข้อมูลผ่านเว็บบราวเซอร์

---

## 🚀 วิธีการรันโปรเจกต์ (Quick Start)

คุณสามารถเปิดใช้งานระบบบนเครื่อง Mac ของคุณได้ง่ายๆ ผ่าน 2 วิธีดังนี้:

### วิธีที่ 1: ดับเบิ้ลคลิกเพื่อเริ่มต้น (ง่ายที่สุดสำหรับ macOS)
1. เปิดโฟลเดอร์โครงการ `AI Agent` ใน Finder
2. ดับเบิ้ลคลิกที่ไฟล์ **`start.command`**
3. ระบบจะเปิดหน้าต่าง Terminal ขึ้นเซิร์ฟเวอร์จำลอง และเปิดเว็บเบราว์เซอร์พาคุณไปยังหน้าแรก (`http://localhost:8000/index.html`) โดยอัตโนมัติ!

### วิธีที่ 2: รันผ่าน Terminal
1. เปิด Terminal และนำทางไปยังโฟลเดอร์ของโครงการ
2. รันคำสั่งต่อไปนี้:
   ```bash
   ./start.sh
   ```

## 💡 ฟีเจอร์หลักของระบบ (Key Features)

### 1. ระบบยืนยันตัวตนและการลงทะเบียน (Authentication & Sign In)
- ผู้ใช้งานทุกคนจำเป็นต้องล็อกอินหรือสมัครสมาชิกก่อนเข้าชมระบบ
- ระบบสมัครสมาชิกรองรับการลงทะเบียนเรียน และสร้างตารางข้อมูลผู้ใช้งาน/แพ็กเกจตั้งต้นให้อัตโนมัติ
- **ข้อมูลสาธิตสำหรับทดลองใช้ (Demo Accounts)**:
  - แอดมิน: `phatcharin@example.com`
  - ครูฝึก (PT): `somchai.pt@example.com`
  - นักเรียน: `kanya@example.com`

### 2. ระบบบทบาทผู้ใช้งาน (Role-Based Control)
- **นักเรียน (Student)**: สามารถใช้ได้เฉพาะหน้าจองคลาสเรียนของนักเรียนเท่านั้น (ซ่อนเมนูสลับบทบาท)
- **ครูฝึก (Trainer / PT)**: จัดการตารางสอนคลาสของตนเอง ซ่อนเมนูสลับบทบาท
- **ผู้ดูแลระบบ (Admin)**: แสดงแผงเมนูสลับบทบาท ("View As") ที่หัวข้อของหน้าจอ เพื่ออำนวยความสะดวกในการกดทดสอบระบบได้ทุกหน้าที่ต้องการ

### 3. ปฏิทินรายเดือนแบบตอบสนอง (Interactive Monthly Calendar)
- แสดงปฏิทินของเดือนปัจจุบันเพื่อความสะดวกในการวางแผนและจองคลาสเรียน (มีในหน้าของนักเรียนและครูฝึก)
- **จุดแสดงตารางเวลา (Calendar Dots)**:
  - นักเรียนจะเห็นจุดระบุสถานะคลาสที่เปิดสอน (จุดสีน้ำเงิน) และจุดระบุคลาสที่ตนเองจองไว้แล้ว (จุดสีเขียว)
  - ครูฝึกจะเห็นจุดสีเขียวในวันที่ตนเองมีคลาสที่ต้องทำการสอน
- **การกรองข้อมูลตามวันที่**: คลิกเลือกวันที่ที่ต้องการบนปฏิทินเพื่อกรองแสดงเฉพาะคลาสของวันนั้น ๆ ได้ทันที และสามารถกดปุ่มล้างตัวกรองได้สะดวก

### 4. กฎการจองและค่าธรรมเนียมคลาสเรียน (Booking Rules & Pricing Tiers)
- **Group Class (คลาสกลุ่ม)**:
  - **เปิดเสรีไม่มีการหักเครดิต**: จองได้ตลอดเวลาโดยไม่ต้องใช้เครดิตคงเหลือ
  - **การทดลองเรียนครั้งแรก**: ราคา 300 NT สำหรับการจองครั้งแรก และราคาปกติ 600 NT สำหรับครั้งต่อ ๆ ไป
  - **เงื่อนไขผู้เข้าร่วมขั้นต่ำ**: ต้องมีคนร่วมคลาสอย่างน้อย 2 คนจึงจะเริ่มคลาสได้ คลาสที่มีคนน้อยกว่า 2 คนจะแสดงสถานะเป็น **"待成班 (Pending)"** และเปลี่ยนเป็น **"已成班 (Confirmed)"** เมื่อมีตั้งแต่ 2 คนขึ้นไปโดยอัตโนมัติ
- **1-on-1 Class (คลาสส่วนตัว)**:
  - 定價方案: 1 คลาส 1800 NT, 10 คลาส 1450 NT/คลาส, 20 คลาส 1350 NT/คลาส
  - หักเครดิตออก 1 ครั้งเมื่อจอง และคืนสิทธิ์ให้เมื่อยกเลิกสำเร็จ

### 5. กฎยกเลิกจองคลาสล่วงหน้า 24 ชั่วโมง (24-Hour Cancellation Policy)
- การกดยกเลิกจองคลาสเรียน (ทั้งคลาสกลุ่มและคลาสส่วนตัว) จะต้องทำล่วงหน้าอย่างน้อย 24 ชั่วโมงก่อนเวลาเริ่มเรียนเท่านั้น
- ระบบจะตรวจสอบเวลาอย่างแม่นยำ หากเหลือน้อยกว่า 24 ชั่วโมง จะไม่สามารถยกเลิกได้เพื่อป้องกันผลกระทบต่อผู้อื่น

### 6. แผงตั้งค่าสำหรับแอดมิน (Admin Settings Panel)
- เพิ่มแท็บ **Settings** ในหน้าของแอดมิน เพื่อให้สามารถแก้ไขราคาคลาสกลุ่ม คลาสเดี่ยว และจำนวนขั้นต่ำในการเปิดคลาสเรียนได้โดยตรงผ่านระบบหน้าเว็บ และอัปเดตไปคำนวณในฝั่งนักเรียนทันทีแบบเรียลไทม์

### 7. ครูฝึกกำหนดเวลาได้เอง และตรวจคาบเวลาทับซ้อน
- ครูฝึกสามารถกำหนดเวลาคลาสเรียนได้เองหลายคลาสต่อวัน
- **ระบบป้องกันตารางเรียนทับซ้อน**: หากครูฝึกทำการลงเวลาสอนซ้ำซ้อนในวันและเวลาเดียวกัน ระบบจะตรวจพบการชนกันของคาบเวลา (Overlap) และแจ้งเตือนข้อผิดพลาดทันที

## 🌐 การนำขึ้นระบบออนไลน์ / โฮสติ้ง (Deployment Guide)

เนื่องจากระบบนี้เป็นเว็บไซต์ประเภท **Static Web Application** (ทำงานบน Browser และบันทึกข้อมูลจำลองไว้ที่ LocalStorage) คุณจึงสามารถนำไปอัปโหลดขึ้นโฮสติ้งฟรีเพื่อใช้งานจริงหรือแชร์ให้ผู้อื่นทดลองใช้งานได้อย่างง่ายดาย:

### 1. โฮสต์ด้วย Netlify (แนะนำ - ง่ายที่สุด)
1. สมัครใช้งานเว็บ [Netlify](https://www.netlify.com/)
2. ไปที่หน้าหลักของ Dashboard เลื่อนลงมายังส่วน **Drag and drop your site folder here**
3. ลากโฟลเดอร์โครงการ `AI Agent` (ที่มีไฟล์ `index.html`, `app.js`, `styles.css`) ไปวาง
4. เสร็จสิ้น! คุณจะได้รับลิงก์ URL (เช่น `https://random-name.netlify.app`) สำหรับส่งให้ผู้ใช้จองคลาสเรียนได้ทันที

### 2. โฮสต์ด้วย GitHub Pages
1. สร้าง Repository ใหม่บน GitHub และอัปโหลดไฟล์โครงการขึ้นไป
2. ไปที่เมนู **Settings** > **Pages** ของ Repository นั้นๆ
3. ในส่วน **Build and deployment** > ให้เลือก Source เป็น **Deploy from a branch** และเลือกสาขาหลัก (เช่น `main`)
4. กดบันทึก ระบบจะสร้าง URL ฟรีให้คุณภายใน 1-2 นาที
